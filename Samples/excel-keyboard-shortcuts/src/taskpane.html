Keyboard Shortcuts

Please sideload your add-in to see app body.

Try the following keyboard shortcuts:

  • Ctrl+Alt+1 to Show task pane.
  • Ctrl+Alt+2 to Hide task pane.
  • Ctrl+Alt+3 to cycle through cell colors.

If using a shortcut that conflicts with existing Excel shortcuts, the user will see a conflict modal. For e.g. try pressing Ctrl+R. The shortcut is programmed to log "test conflict" in the console, but in order to use the shortcut, the user must first confirm which action the shortcut must take. After making a selection, if you wish to reset the shortcut preference, search for "Reset Office Add-in Shortcut Preferences" in the Tell Me Search box.
